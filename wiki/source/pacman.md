# مدیر بسته‌ی Pacman در آرچ لینوکس

`pacman` ابزار مدیریت بسته پیش‌فرض در آرچ لینوکس است که وظیفه‌ی نصب، به‌روزرسانی، حذف و مدیریت بسته‌ها را بر عهده دارد. این ابزار با زبان C نوشته شده و به سادگی و سرعت بالا مشهور است.

---

## ویژگی‌ها

- 📦 **ترکیب نصب و مدیریت بسته‌ها در یک ابزار**
- 🔄 **پشتیبانی کامل از بسته‌های باینری و ساخت از سورس**
- 📁 **ردیابی وابستگی‌ها و مدیریت آن‌ها به‌صورت خودکار**
- 🧩 **هماهنگ با مخازن رسمی آرچ و مخازن شخصی**

---

## دستورهای رایج

### 📥 نصب بسته
```bash
sudo pacman -S <package-name>
```
مثال:
```bash
sudo pacman -S firefox
```

### 🗑️ حذف بسته
```bash
sudo pacman -R <package-name>
```
حذف همراه با فایل‌های وابسته‌ی بی‌استفاده:
```bash
sudo pacman -Rns <package-name>
```

### ♻️ بروزرسانی کل سیستم
```bash
sudo pacman -Syu
```
- `S` → همگام‌سازی پایگاه داده
- `y` → به‌روزرسانی لیست بسته‌ها
- `u` → ارتقای بسته‌های نصب‌شده

### 🔍 جستجوی بسته
```bash
pacman -Ss <search-term>
```

### 🔍 جستجو در بسته‌های نصب‌شده
```bash
pacman -Qs <search-term>
```

### 📄 لیست فایل‌های نصب‌شده توسط یک بسته
```bash
pacman -Ql <package-name>
```

### 📦 بررسی اینکه یک فایل متعلق به کدام بسته است
```bash
pacman -Qo /path/to/file
```

---

## مدیریت کش (Cache)

بسته‌ها پس از نصب در کش `/var/cache/pacman/pkg/` باقی می‌مانند. برای پاکسازی:

### 🧹 حذف بسته‌های قدیمی:
```bash
sudo pacman -Sc
```

### 🧨 حذف کامل تمام کش (به جز نصب‌شده‌ها):
```bash
sudo pacman -Scc
```

---

## رفع خطاهای رایج

### 🔒 فایل قفل شده
اگر هنگام اجرای pacman خطای زیر را دیدید:
```
error: failed to initialize alpm library
(could not lock database)
```
علت معمول: اجرا شدن همزمان دو فرایند `pacman`. راه حل:
```bash
sudo rm /var/lib/pacman/db.lck
```

---

## تعامل با AUR

Pacman به‌صورت پیش‌فرض به مخزن AUR دسترسی ندارد. برای استفاده از AUR، ابزارهای کمکی مانند `yay` یا `paru` توصیه می‌شوند.

---

## جمع‌بندی

`pacman` ابزاری قدرتمند، سریع و منعطف برای مدیریت بسته‌ها در آرچ لینوکس است. یادگیری آن یکی از قدم‌های کلیدی برای کنترل بهتر سیستم در آرچ محسوب می‌شود. با ترکیب قدرت `pacman` و مخزن گسترده AUR، آرچ لینوکس به یکی از انعطاف‌پذیرترین و قابل‌کنترل‌ترین توزیع‌های لینوکس تبدیل شده است.

> "pacman is simple — just like Arch itself."
